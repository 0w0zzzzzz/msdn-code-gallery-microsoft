How to use MongoDB with ASP.NET Core

Introduction

MongoDB is a popular NOSQL database, and it can work on ASP.NET Core.

This sample will demonstrate how to use MongoDB in ASP.NET Core.

Sample prerequisites

  • .NET Core 1.0 or later version(s). [.NET Core + Visual Studio tooling]
  • Microsoft Visual Studio 2015 update3 or above. [Visual Studio 2015 installer]

Building the sample

• Configure the settings of sample.

• Open the appsettings.json of project,

• Local to ConnectionString field, and fill the value as your MongoDB connection string.

• If your MongoDB is neither at Azure nor ursing the ssl, you should change the IsSSL as False.

• Restore the packages.

• Right click on project, and select Restore Packages.

• Press F6 Key to build the sample project.

Running the sample

• Use Visual studio to open the sample solution, then press F5 Key or select Debug -> Start Debugging from the menu.

• When debug is running, you can see this:

 

Click Add to add page and fill all fields.

 

Click Add button.

The new data will be created and then be back to list page.

For build new project

• Create a new ASP.NET Core project

• Right click on the project and select Manage NuGet Packages

• In the browser tab, type MongoDB in search box.

• Select MongoDB.Driver, ensure the version is 2.3 or later. Click Install.

Using the code

The MongoDbContext

 

C#
Edit|Remove
csharp 
    public class MongoDBContext
    {
        public static string ConnectionString { get; set; }
        public static string DatabaseName { get; set; }
        public static bool IsSSL { get; set; }

        private IMongoDatabase _database { get; }

        public MongoDBContext()
        {
            try
            {
                MongoClientSettings settings = MongoClientSettings.FromUrl(new MongoUrl(ConnectionString));
                if (IsSSL)
                {
                    settings.SslSettings = new SslSettings { EnabledSslProtocols = System.Security.Authentication.SslProtocols.Tls12 };
                }
                var mongoClient = new MongoClient(settings);
                _database = mongoClient.GetDatabase(DatabaseName);
            }
            catch (Exception ex)
            {
                throw new Exception("Can not access to db server.", ex);
            }
        }

        public IMongoCollection<Post> Posts
        {
            get
            {
                return _database.GetCollection<Post>("Posts");
            }
        }
    }
