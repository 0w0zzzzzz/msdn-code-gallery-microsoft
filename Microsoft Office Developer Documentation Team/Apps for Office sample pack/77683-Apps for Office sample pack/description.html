Old description:  

 

The apps for Office platform lets you create engaging new consumer and enterprise experiences that run within supported Office 2013 applications. An app for Office is basically a webpage that is hosted inside an Office application and can extend the functionality of a document, email message, meeting request, or appointment. Apps for Office combine cloud services with standard web technologies such as HTML5, CSS3, JavaScript, XML, and REST.

The code samples in this sample pack demonstrate the following types of apps:

  • Task pane apps for Excel 2013, PowerPoint 2013, Project
    Professional 2013, and Word 2013.
  • Content apps for Excel 2013 and Excel Web App.
  • Mail apps for Outlook 2013 and Outlook Web App.

This sample pack includes all of the code samples developed for apps for Office. You can download a single ZIP file that contains all of the samples by choosing one of the Download buttons. You can also download the individual samples or read more about them by choosing the View samples tab.

For more information about how to start developing apps for Office, see the Apps for Office and SharePoint Dev Center. For in-depth developer documentation, see Build apps for Office.
